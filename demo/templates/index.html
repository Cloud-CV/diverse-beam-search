Diverse Beam Search

Decoding Diverse Solutions from Neural Sequence Models

{% endblock %} {% block slider %} {% endblock %} {% block abstract%}

Beam search, the standard work-horse for decoding outputs from neural sequence models like RNNs produces generic and uninteresting sequences. This is inadequate for complex AI tasks like image-captioning that are inherently ambiguous.
To overcome this, we propose a diversity-promoting replacement, Diverse Beam Search that produces sequences that are significantly different — with runtime and memory requirements comparable to beam search.
